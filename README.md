# Go Simple Rest API

Used for trainings in CI/CD and DevOps.

## To run the application

```shell
go run simpleRest.go
```

## To format the code

```shell
go fmt simpleRest.go
```

## To check style

```shell
golint
```

## To check suspicious constructs

```shell
go vet
```
